解决Jekyll无法自动生成问题

2013-04-26

Jekyll 的自动生成功能还是很有用的，不用每次都重新执行一次 jekyll --server 通过google发现也有人和我一样的问题

在stack overflow 有人给出了解答 Jekyll's “--auto” doesn't work?

sudo gem uninstall directory_watcher && sudo gem install directory_watcher -v 1.4.1

我才发现原来我装了两个版本的 directory_watcher, 其中一个是1.5的

于是通过gem将1.5版本的 directory_watcher 删除之后Jekyll就正常了。

所以如果大家的 自动生成功能失效， 请注意下 directory_watcher 是否只保留了1.4.1的版本。
