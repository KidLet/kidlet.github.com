JavaScript学习笔记 （2）类型，值，变量

计算机通过操作“值”（value）来进行工作， 例如 数字3.14, 文本“Hello World”。可以被计算和进行操作的“值”的种类叫类型（type）。

当计算机需要保留一个值便于以后使用， 值可以被 赋值（或者说存储）到“变量”（variable）中。变量定义了它的名字，并且可以通过名字来操作它所存储的值。

JavaScript 类型可以分为两大类：自有类型（primitive types）， 对象类型（object types）

1.自有类型包括：数字，字符串，布尔值。

2.对象类型：除了自有类型剩下的就是对象类型，主要有，数组，函数，正则表达式，自定义对象等。

数字 （Numbers）

与其他语编程言不同的是， JavaScript并没有区分 整数和浮点数， 所有的数字在JavaScript中都是用64位浮点的方式保存的，采用IEEE 754标准。 值得注意的是， 在数组的索引（下标）和位运算的操作符的整数是采用32位整数表示的。

直接出现在 JavaScript代码中的数字 叫做 数值文字（numeric literal）。

整数表示

对于要表示十进制整数，可以在代码中直接编写，非常符合自然阅读习惯。

JavaScript支持十六进制表示， 只要在前面加上 “0X” 或者 “0x”， 后面是 十六进制整数数字就可以了

0xff // 15*16 + 15 = 255 (base 10)
0xCAFE911

JavaScript对以前导“0”来表示八进制数的支持不是很好，不推荐使用，并且在ECMAScript 5 中已经明确禁止了这种方法。

浮点数表示

这里的表示和 C，JAVA等类似 下面是例子，看完例子也就明白了。

3.14
2345.789
.333333333333333333
6.02e23     // 6.02 × 10^23
1.4738223E-32    // 1.4738223 × 10^−32

当然更正规的表示如下：[digits][.digits][(E|e)[(+|-)]digits]

字符串

JavaScript字符串的定义是：一串有序的不可变的 16位Unicode值。字符串的长度是它所包含的16位Unicode字符的数量。字符串也是数组，下标也是从0开始。空字符串的长度为0。JavaScript没有特定的一种类型来表示字符，如果要表示单个字符，可以使用长度为1的字符串。

字符串表示

在代码中要表示字符串， 可以简单的用单引号（'）或者双引号（"）将文本括起来。下面是一些字符串表示的例子

"" // The empty string: it has zero characters
'testing'
"3.14"
'name="myform"'
"Wouldn't you prefer O'Reilly's book?"
"This string\nhas two lines"
"π is the ratio of a circle's circumference to its diameter"

字符串转义

有时候要在字符串中表达一些无法输入或者会引起歧义的字符，那么常常需要借助转义符反斜杠“\”

例如：'You\'re right, it can\'t be a quote' 它所表示的字符串为：You're right, it can't be a quote

可以看到反斜杠后面跟着的单引号并不是 字符串的界定符，而是一个实在的文本表示 下面是一些转义的规则

\0 空字符 (\u0000)
\b 空格 (\u0008)
\t 水平制表符 (\u0009)
\n 换行 (\u000A)
\v 垂直制表符 (\u000B)
\f 换页符 (\u000C)
\r 回车符 (\u000D)
\" 双引号 (\u0022)
\' 单引号 (\u0027)
\\ 反斜杠 (\u005C)

简单使用字符串

JavaScript有个特性是可以随意地连接（concatenate）字符串， 利用 “+” 操作符。当 “+” 用在数值类型时，代表将两个数相加。而在字符串运算时，代表将两个字符串连接在一起。

msg = "Hello, " + "world"; // Produces the string "Hello, world"
greeting = "Welcome to my blog," + " " + name;

要获取某个字符串的长度可以用 length 属性

name="kidlet";
name.length;    // 6

下面是一些基本使用

var s = "hello, world"
s.charAt(0) //返回第一个字符
s.charAt(s.length-1) //返回最后一个字符
s.substring(1,4)    //返回从 第二个字符到第四个字符形成的子串
s.slice(1,4)
s.slice(-3)
s.indexOf("l")      //返回第一次出现“l”的位置
s.lastIndexOf("l")  //返回最后一次出现“l”的位置
s.indexOf("l", 3)   //返回从第三个字符开始第一次出现“l”的位置

正则表达式

脚本语言基本上都有个特点是 支持正则表达式，用于匹配、替换文本等等。在很大程度上简化程序。 在JavaScript中可以利用， RegExp() 来创建RegExp对象，它的正则表达式语法与Perl保持一致

下面是一个例子：

var text = "testing: 1, 2, 3";
var pattern = /\d+/g    //匹配任意的数字
pattern.test(text)      //true， 存在匹配
text.search(pattern)    //9,第一个匹配所在的位置
text.match(pattern)     //["1", "2", "3"]，所有匹配的数组结果
text.replace(pattern, "#"); //"testing: #, #, #"，将所有匹配成功的数字替换为 #

布尔值 （Boolean Values）

一个布尔值代表着 真或者假，开或者关，对或者错。对于布尔类型来说，只存在两个值，它们分别被用保留字
true 和 false 来表示。

布尔值通常都被用来作为比较返回的结果

以下的的值都可以被转换为 false

undefined
null
0
-0
NaN
"" // 空串

除此以外的其他值均可以转换为 true

布尔值有一个 toString() 方法，可以将布尔值转为 “true” 或者 “false”字符串

并且还有三个运算符可以进行布尔值运算，分别是 &&运算符 代表布尔与运算， ||运算符 代表布尔或运算

!运算符 代表布尔非运算

if ((x == 0 && y == 0) || !(z == 0)) {
// x 和 y 都是0 或者 z 不是0
}

null 和 undefined

null是一个关键字，用来表示缺少值，空值的意思。同时也表示一个特殊对象值“no object”。它的含义类似其他语言中的null或者ruby中的nil。

JavaScript还有另外一个特殊值用来表示更深层含义的 “值不存在” -- undefined

undefined有两种含义：

  1. 变量的值未初始化
  2. 对象的属性不存在或者数组的元素不存在
  3. 函数没有返回值
  4. 没有为函数的参数（parameter）提供实参（argument）

null和undefined被 == 运算符认为是一样的 但是可以使用 === （strict equality operator）来区分他们

null 和 undefined 的理解

可以理解null表示是 程序级别的（program-level），正常的，预期的缺少值。

undefined表示的是 系统级别的（system-level），异常的，错误的缺少值。

如果需要传递参数，赋值一个“缺少值”的含义等操作，选择null通常是正确选择。

变量 （Variable）

在使用一个变量的时候，需要先声明（declare）它，变量的声明采用关键字 var，下面是例子：

var i;
var sum;

也可以同时声明多个变量，通过逗号分隔开(,)

var i, sum;

在声明变量的时候，同时也可以为变量初始化，就像下面这样。

var message = "hello";
var i = 0, j = 0, k = 0;

如果不为变量初始化，那么默认变量为undefined

JavaScript声明变量的时候无需指出变量的类型，因为JavaScript变量可以容纳任意类型的值

var i = 10;
i = "ten";
